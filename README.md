<br/>

> An end-to-end deep learning image classification system for real-time detection of cucumber growth stages, designed to modernize agricultural monitoring through AI-powered inference.

<div align="center">

| Attribute | Details |
|:---|:---|
| **Stack** | Python · TensorFlow · Keras · NumPy · Pandas · Matplotlib |
| **Domain** | Computer Vision · Agricultural AI · Deep Learning |

| **Performance** | Accurate stage prediction across all defined agricultural growth stages |
| **Dataset** | Custom-curated and hand-labelled agricultural image dataset |
| **Security** | On-premise model inference, no external data exposure |
| **Impact** | Replaces manual crop monitoring with automated, scalable AI inference |

**Engineering Summary:**

Designed and implemented a production-grade CNN classification pipeline from first principles. The system ingests raw field imagery, applies structured preprocessing and data augmentation, then routes through a multi-layer convolutional architecture for accurate growth-stage classification. Model training, validation, and cross-stage evaluation were conducted with rigorous performance benchmarking. Dataset curation, labelling, and pipeline construction were executed in-house. The solution is architected for real-world deployment in agricultural monitoring environments, offering a direct upgrade path from manual inspection to AI-driven field intelligence.
